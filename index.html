Hermann is a minimal friction time tracker for macOS and iOS

Тут фичи с картинками по конкретным фичам

  • Быстрый запуск таймера без трения. Поле поиска рядом над пальцем в iOS.
  • Быстрый запуск таймера без трения. ФОрма быстрого создания на macOS
  • Быстрый запуск таймера без трения. ML-подсказка при выборе задачи
  • Возможность вести задачи по GTD, при учёте, что задач за год может скопиться несколько тысяч
  • Показывает текущую задачу в менюбаре. может скрывать её, если экран шарится.
  • Не пиздит данные, всё в айклауде и у разработчика нет доступа к этим данным.
  • Быстрое двигание старта задачи
  • Синхронизация между macOS и iOS

Тут что не умеет

  • Работать на виндоуз, андроиде и вебе
  • Учёт заработанных денег по проектам. Что не делает? Не ведёт подсчёт денег. Герман не про деньги, он про смысл жизни.
  • Различие времени потраченного на оплачиваемые и не оплачиваемые проекты.
  • Учёт времени на команду работников.
